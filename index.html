Programmation des modules

Module Relais

Commutation ON/OFF pour éclairages, prises et équipements. Contrôle jusqu'à 24 circuits indépendants avec contacts secs.

Module Variateur

Contrôle graduel de l'intensité lumineuse. Compatible avec LED dimmables, halogènes et incandescences avec courbes adaptatives.

Module IP Port

Passerelle entre votre installation TIS Bus et réseau IP. Permet accès distant, contrôle par application et intégration aux systèmes tiers.

Module 4 Zones

Contrôleur multifonction gérant 4 zones distinctes. Combine relais, entrées numériques pour automatiser des pièces entières.

Schémas de câblage

TIS BUS - Guide de câblage

Le TIS BUS utilise un code couleur spécifique pour faciliter l'installation:

  • Marron: +12V DC
  • Orange: GND (Masse)
  • Bleu: DATA -
  • Vert: DATA +

Configuration en topologie Bus ou Étoile possible.

Voir le guide complet

Informations techniques

  • Distance maximale: 1000m
  • Section de câble recommandée: 0.75mm²
  • Résistance terminale: 120Ω
  • Tension de fonctionnement: 12V DC
